Software Development Engineer Intern

As an intern at Amazon on the Kindle Content Delivery Services team, I was responsible for streamlining the operational workflow and enhancing the security of resources. I researched and implemented an automation solution for the URL signing process using AWS Cloud Development Kit, AWS Secrets Manager and AWS Lambda. As a result of this implementation, the URL signing process was completely automated, saving an estimated six developer hours every 90 days. This automation also increased the overall security of the resources.

Dow Jones & Company

Software Engineer Intern

As an intern at Dow Jones & Company on the Factiva content management team, I was tasked with creating an application to organize all of the news stories received by Factiva and give the team a simple way to search and download the original news files. So I built an application that utilized AWS S3 to store the over 100,000 news stories per day and a web UI using ASP.NET that gave the team the ability to filter through the files using various metadata. As a result of my work, the team now has a reliable application that they can use to find and interact with specific news files, improving the overall workflow.
